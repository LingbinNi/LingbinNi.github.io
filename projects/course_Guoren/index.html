Guoren - A Social Site for UCAS

A social platform for students in UCAS under the Ruby on Rails framework


This is the teamwork of the course Advanced Software Engineering, which is completed by Jingqiao Zhu, Yu Zhang, Jixian Yin, Fangzhou Bai, and Lingbin Ni. Given that there are currently no specialized social forum websites for students at UCAS, the initial aim for this project is that we hope to establish such an exchange platform to meet students' needs for learning and communication, activity groups, lost and found items, notices, and announcements.

Technological implementation

Front end
  • Layout: HTML+CSS+Bootstrap framework
  • Communication: JavaScript+Jquery (Ajax)
Back end
  • Ruby on Rails
Database
  • PostgreSQL (Ruby Model Interaction)
Third party plugin tools
  • Text Editor, etc.

Data preprocessing

As a first step, I converted the images to grayscale because grayscaling removes clutter from an image. Since the problem includes only only classifying images, grayscaling allows the feature maps to concentrate only on the subject under interest. Also, grayscaling converts a 3-channel RGB image to a single channel image which reduces the computation complexity.
