Multi-Omics Database Development


An integrated soybean multi-omics database named SoyOmics of genomic, transcriptomic, epigenomic, and phenotypic data of diverse accessions.


Overview


Overview of SoyOmics


Conclusions

  • SoyOmics integrated soybean multi-omics datasets including genomic, transcriptomic and epigenomic data from different tissues, organs and accessions.
  • By equipping with multiple analysis modules and toolkits, SoyOmics is of great utility to facilitate the global scientific community to fully use these big omics datasets.
